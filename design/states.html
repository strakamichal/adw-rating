Empty, Error & Loading States

Design patterns for when content is loading, empty, or something goes wrong.

State 1

No Results (Empty Filter)

Shown when the user's filter combination returns no results. Includes a clear message and a call-to-action to recover.

🔍
No teams match your filters
Try broadening your search or removing some filters to see more results.
State 2

404 — Page Not Found

Shown when a user navigates to a URL that does not exist (e.g., a deleted team profile or a typo in the URL).

Page Not Found
The page you are looking for does not exist or may have been moved. Check the URL for typos, or use the links below to get back on track.

State 3

Loading Skeletons

Skeleton placeholder blocks shown during initial page load, matching the shape of expected content. The shimmer animation communicates that content is loading.

Error State
