# Category order should be Large - Intermediate - Medium - Small

- **Type**: improvement
- **Priority**: medium
- **Status**: open

## Description

Categories are displayed in an inconsistent or incorrect order throughout the app. The correct order everywhere should be: Large, Intermediate, Medium, Small (from largest to smallest).

## Where to look

- Any page that lists or filters by dog size category (rankings, competitions, competitor profiles, etc.)
- Domain enum or display logic for `DogSize` / category ordering

## Acceptance criteria

- [ ] Categories appear in order Large - Intermediate - Medium - Small everywhere in the UI
