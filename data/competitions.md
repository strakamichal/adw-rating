## SmarterAgility API

Public JSON API, no authentication required. Primarily designed for live results, but also returns completed competitions.

### Endpoints

```
# 1. List competitions
GET https://www.smarteragility.com/sal/competitions/?scope={scope}&_={timestamp}
#    scope: recent | past | upcoming | all | find
#    For search (scope=find): &year=2024&month=6&title=open

# 2. Competition detail (list of rounds)
GET https://www.smarteragility.com/sal/competition/{hash_code}?_={timestamp}

# 3. Round results
GET https://www.smarteragility.com/sal/round/{round_hash}?_={timestamp}
```

### Download procedure

1. Find competition via `/sal/competitions/` → get `hash_code`
2. Load rounds via `/sal/competition/{hash_code}` → array `rounds[].hash`
3. For each round, download results via `/sal/round/{round_hash}`

### Run data structure

| Field | Type | Description |
|-------|------|-------------|
| `dorsal` | string | Bib number |
| `handler` | string | Handler |
| `dog` | string | Full registered name |
| `dog_short` | string | Call name |
| `breed` | string | Breed |
| `country_name` | string | Country |
| `team_name` | string | Team |
| `ranking` | int | Placement |
| `course_time` | string | Course time |
| `time_faults` | string | Time faults |
| `course_faults` | int | Course faults |
| `refusals` | int | Refusals |
| `total_faults` | string | Total faults |
| `speed` | float | Speed (m/s) |
| `is_eliminated` | int | Eliminated (0/1) |
